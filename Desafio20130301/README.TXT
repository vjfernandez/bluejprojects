Solución al pequeño desafío del viernes:

 Construir un método   
     public int[] invertirTirandoLosNegativos(int[] n)
     
 que devuelva un array con los mismos números que se pasan en el array n
 pero en orden inverso y descartando los negativos.
 
 Recuerda que para devolver un array es necesario saber cuántos elementos
 tendrá.
 Es necesario 
    1) contar los números no negativos de n,
    2) Crear un array para el retorno, con new
    3) Rellenar el array de retorno
